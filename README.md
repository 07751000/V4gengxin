使用方法：
将直播源地址直接放到 播放器中即可。地址会每天自动更新最新的源，不需要人工干预。千万不要把地址里面的内容先用浏览器访问 获得内容然后再自己保存到txt中，这样虽然可以播放，但是以后就无法自动获得最新的更新了。。。。。。
